# JDB 客户端产品规范

> 日期：2026-07-21  
> 来源：项目设计需求、当前 Flutter 实现、`docs/main/api/jdb_api_openapi.json`、`RULES.md`

## 1. 总体规则

- 应用定位：JavDB 第三方客户端，实现官方客户端除付费、在线观影、广告外的主要浏览与管理能力。
- 技术栈：Flutter、Material 3、go_router、provider、dio、json_serializable。
- 文案：按项目约定直接使用中文硬编码，不接入 l10n/ARB。
- 结构：继续使用 Feature-First，业务页位于 `lib/features/<feature>/screens/`，跨模块复用组件放在 `lib/core/widgets/`。
- 动态域名：启动、持久化、608/失败轮转、重试逻辑以 `docs/main/api/api-reference.md` 为准。
- 列表体验：排行榜、类别、演员列表、搜索结果、我的内容页均按分页/瀑布式加载实现；首页固定区块可使用 3x3 数据切片。

## 2. 主导航

底部导航保留 5 个入口：

| Tab | 路由 | 页面 |
| --- | --- | --- |
| 首页 | `/home` | `HomePage` |
| 排行榜 | `/rankings` | `RankingsPage` |
| 类别 | `/categories` | `CategoriesPage` |
| 演员 | `/actors` | `ActorsPage` |
| 我的 | `/profile` | `ProfilePage` |

详情、搜索、通用列表、设置与个人内容页使用顶层路由，不显示为独立底部 Tab。

## 3. 首页

- 顶部豆腐块横向滚动，入口为：看热播、AV资讯、看短评、找磁链、识演员、识影片、系列、片商、导演。
- `佳片推荐`：右侧 `往期推荐`，主体使用轮播图展示图片和标题。
- `最新上架`：右侧 `全部`，主体为 3x3 影片卡片网格，支持 `换一组`。
- `近期磁链更新`：右侧 `全部`，主体为 3x3 影片卡片网格，支持 `换一组`。
- 影片卡展示封面、标题、番号；点击进入 `/movie/:id`。

## 4. 排行榜

顶部 Tab：`Top250`、`看热播`、`有码`、`无码`、`欧美`、`FC2`。

- `Top250` 需要登录；未登录展示登录引导。页面顶部有筛选按钮，列表项使用三行纵向布局：封面/截图与排名角标、标题、番号/日期/备注。
- `看热播` 顶部两个单选组合：`高评价/全部` 与 `日榜/周榜/月榜`，默认 `高评价` + `日榜`。
- `有码`、`无码`、`欧美` 顶部单选组合：`日榜/周榜/月榜/演员月榜`。
- `FC2` 顶部单选组合：`日榜/周榜/月榜`。
- 非 Top250 内容使用影片卡片网格。

## 5. 类别

顶部第一排 Tab：`有码`、`无码`、`欧美`、`FC2`、`动漫`。

第二排工具栏左侧为排序下拉，右侧为筛选抽屉按钮。不同 Tab 的排序和筛选选项可以不同，最小实现需按 Tab 提供独立 schema。

内容区使用影片卡片网格，展示封面、标题、番号。

## 6. 演员

顶部 Tab：`推荐`、`有码(女)`、`有码(男)`、`无码`、`欧美(女)`、`欧美(男)`。

- `推荐` 不做瀑布式加载，分区为 `新人 - 更新日期`、`月排名`、`Fanza(DMM)推荐 - 更新日期`。
- 其他 Tab 使用演员卡片网格；`有码(女)` 顶部带筛选按钮与筛选抽屉。
- 演员卡展示头像与名称；点击进入 `/actor/:id`。

## 7. 我的

未登录时只展示大登录按钮和设置 cell。登录后展示用户卡片和以下 cell：

- 我想看的：小标题 `我想看 X 部影片`
- 我看过的：小标题 `我已看过 X 部影片`
- 我的关注
- 我的收藏
- 我的清单
- 近期浏览
- 个人资料
- 设置

子页面规则：

- `我想看的`、`我看过的`：顶部 Tab 为 `全部/有码/无码/欧美/FC2/动漫`，右侧筛选按钮，影片网格分页。
- `我的关注`：顶部 Tab 为关注列表，列表项使用影片三行布局。
- `我的收藏`：cell 列出收藏的演员、片商、系列、导演、番号、清单，并进入对应子页面。
- `个人资料`：cell 展示电子邮箱、短评被举报次数、短评被删次数、禁言次数、待审核/已通过订正数、修改密码、修改用户名。
- `设置`：cell 展示外观模式、线路选择、默认筛选标签、清除缓存。

## 8. 详情页

影片详情 `/movie/:id`：

- 顶部标题加粗，超长自动省略。
- 主体展示封面、番号、发行日期、时长、导演、片商、系列、评分、想看/看过/存入清单按钮组、想看/看过人数、类别、演员、剧照、TA还出演过、你可能也喜欢。
- 底部悬浮抽屉 Tab：`磁链下载`、`短评`、`相关清单`。

演员详情 `/actor/:id`：

- 展示头像、姓名、出演过影片数。
- `更多信息` 打开抽屉，展示姓名、影片数、生日、年龄、身高、罩杯、胸围、腰围、臀围、出生地。
- 内容区使用影片卡片网格分页。

## 9. 搜索与通用页

搜索页：

- 初始态包含搜索框、近期热搜、历史搜索。
- 结果态 Tab：`影片`、`演员`、`系列`、`片商`、`导演`、`清单`、`番号`。
- 影片与演员使用瀑布式卡片网格；其余使用 cell 列表。

通用影片列表页：

- 导航栏显示标题。
- 顶部左侧单选组合：`全部`、`可播放`、`含磁链`、`字幕`，默认 `含磁链`。
- 右侧排序下拉。
- 内容区为影片卡片网格分页。
